## Troubleshooting

**Old header still showing?**
- Clear browser cache (Cmd+Shift+Delete)
- Clear Hostinger LiteSpeed cache
- Verify files in /public_html/ (not subdirectory)
- Wait 5 minutes and reload

**Domain URL still visible?**
- Confirm mu-master-cleanup.php uploaded
- Clear all caches
- Use incognito/private browsing to test
- Check file permissions (644)

**Navigation not showing?**
- Verify mu-cairde-header-complete.php uploaded
- Clear cache and reload
- Check browser console (F12) for errors

**Styling broken?**
- Clear all caches completely
- Wait for LiteSpeed cache to refresh
- Check both files are uploaded
- Try incognito window
